# CozyLibrary - Data Cleaner

Servicio para procesar y limpiar datos de Google Books API.

##  Repositorios Relacionados
- [Backend API (Java)](https://github.com/tu-usuario/cozy-library-backend)
- [Frontend (Angular)](https://github.com/tu-usuario/cozy-library-frontend)

##  Integración
Este servicio recibe datos de Google Books API y los envía limpios al Backend.
